### Magent AXIOM Module 4 Overview

## Objective
The objective of this assignment was to complete the Magnet AXIOM module 4. It was also done to help develop my overall digital forensics skills. 

## Skills Gained
- Interpreting NTFS (Windows) file systems.
- Identifying and analyzing key artifacts.
- Creating and keeping a chronological timeline of evidence.
- Gaining a better understanding of the Magnet AXIOM tool.

## Required Flags 

1.	What web site did the user download the program from and what is the end date/time of the download? 
2.	What web browser was used to download it? 
3.	How did the user locate the web page for download and what search engine was used? 
4.	When was the last time that the program Ccleaner64.exe was run? 
5.	Where is the program TOR saved on the computer? 
6.	Has TOR ever been run?  If so, when was the last time it was run? 
7.	What search activity is there on the computer five minutes prior to and after TOR was run? 
8.	At there any .onion Tor URL’s on the computer?  If so, can you tell the type of web page?
